### Install
```smalltalk
"Loads the Dawn theme in Pharo 11 (current latest, Feb 2024)"
Metacello new 
	baseline: 'PharoDawnTheme';
	repository: 'github://sebastianconcept/PharoDawnTheme:latest';
	load.	
```
